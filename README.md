# मतदान नेपाल — Matadan Nepal

Live election results platform for Nepal's 2082 BS / 2026 AD House of Representatives election.

**Live site:** [matadan-nepal.vercel.app](https://matadan-nepal.vercel.app)

## Features
- Real-time dashboard: 275 seats (165 FPTP + 110 PR)
- Constituency detail pages for 27 key races
- PR seat tracker with Webster method calculator
- Countdown to Falgun 21, 2082 BS (5 March 2026)
